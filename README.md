# File Organizer Python Project

This project automatically organizes files into folders.

## Features

- Sort Images
- Sort Documents
- Sort Videos
- Sort Programs
- Auto Folder Create

## Technologies

- Python
- OS Module
- Shutil Module

## How to Run

1 Install Python

2 Run command:

python file_organizer.py

3 Enter Folder Path

Example:

C:\Users\Downloads
